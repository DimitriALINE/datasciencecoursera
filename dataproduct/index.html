Choose Player

The app uses the following function named listplayer. This function give the complete list of the players with a given Team and a given Position. The Team and Position are the two arguments of this function.

listplayer <- function(team,position){
  team <- as.character(team)
  position <- as.character(position)
  player <- NBA[(NBA$Team == team) & (NBA$Position == position),]  
  player <- player$Name  
  player  
}

Here, you can see an exemple with the team Boston and the position Point Guard.

listplayer("BOS","PG")
